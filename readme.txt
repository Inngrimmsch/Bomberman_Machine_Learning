  trashy_models:
  
  in this folder are all dictionaries that resulted from the training of the first q_learning-agent.

  simple_q_learning_corrected:
  
  In this folder are all dictionaries and all code that was used in the training of the second q_learning-agent.

  newest_deep_Q_learning_agent:

  This folder contains the code of the newest/best two deep Q_learning agents.

  all_deep_Q_learning_agents:

  This folder contains the code of all the agents we talked about in the report.
